# 🧠 안면 인식 웹 애플리케이션 (Face Recognition Web App)

이 프로젝트는 FastAPI, React, MediaPipe, OpenCV를 활용한 **웹 기반 안면 인식 시스템**입니다.  
사용자는 웹 브라우저를 통해 얼굴을 등록하고, 실시간으로 인식 결과를 확인할 수 있습니다.
